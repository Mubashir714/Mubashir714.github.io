**Technologies:** Python, Jupyter Notebook, Colab, NumPy, Pandas, Matplotlib, PIL

Designed a reproducible experiment to compare **static and dynamic visual encodings** for angle perception, investigating how the way data is visually represented affects a viewer's ability to accurately judge angles.

**Key contributions:**
- Implemented ground-truth extraction pipelines to establish accurate baseline measurements
- Built dynamic frame generation to create animated visual encodings for comparison against static counterparts
- Designed a custom evaluation framework based on **MLAE (Mean Log Absolute Error)** to quantitatively measure perception accuracy across encoding types

This project contributes to the broader field of **visualization research**, helping inform design choices for charts and graphics where angle-based encoding is used to communicate data.
