**Technologies:** Oracle Apex, SQL

Built as the winning submission at the **UN World Innovation Day Hackathon 2023**, VerifAid is a **donation management system** designed to help NGOs securely manage donor request data and streamline charitable operations.

**Key contributions:**
- Designed and implemented a relational database structure to securely manage donor records and donation requests
- Wrote SQL queries to create tables, update database records, and generate visualizations of donation activity for NGO administrators
- Built the application using Oracle Apex, enabling rapid development of a functional, deployable system within hackathon time constraints

This project earned recognition as the **Winner of the UN "World Innovation Day Hackathon 2023,"** organized by Hackmaker, for its practical approach to improving transparency and security in NGO donation management.
