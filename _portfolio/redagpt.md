**Technologies:** LangChain, Auto-GPT, Redis database

Developed as part of the **LangChain x Autonomous GPT Agents Hackathon**, RedAGPT is a novel AI framework [(research proposal)](https://drive.google.com/file/d/1mKLMbjHU6BAj02PvrLE7etEluZDRadSt/view?usp=sharing) utilizing AutoGPT and LangChain for **autonomous vulnerability assessment**.

**Key contributions:**
- Designed an autonomous agent architecture capable of identifying and assessing security vulnerabilities with minimal human intervention.
- Engineered an interactive evaluation platform to facilitate systematic study of AI-driven security assessment approaches.
- Used Redis as the underlying data layer to support agent state management and fast retrieval during autonomous operation.

This project explored how autonomous LLM agents can be applied to **cybersecurity workflows**, contributing a research-oriented framework rather than a production tool — aimed at advancing systematic study in this emerging area.
